# LearningProject

前端知识体系学习与面试复习项目。Vue 3 + TypeScript + Vite，纯前端，零外部 UI 库。

覆盖 JS 基础 → Vue 3 深度 → 浏览器与网络 → 工程化 → 扩展，每个知识点达到原理层/手写层/对比层/源码层深度。

## 已完成页面

| 路由 | 内容 |
|------|------|
| `/` | 学习路线首页，分组展示 + 进度统计 |
| `/html-elements` | 块级/行内元素（7 章 + 9 道面试题） |
| `/web-worker` | Web Worker 原理 + 大厂案例（10 道面试题） |
| `/javascript-core` | 作用域与闭包 + 执行上下文（17 道面试题） |
| `/javascript-this` | this 四种绑定规则 + 箭头函数（6 道面试题） |
| `/javascript-prototype` | 原型与原型链 |
| `/javascript-event-loop` | 事件循环机制 |
| `/javascript-promise` | Promise 原理与手写 |
| `/javascript-handwriting` | 手写题合集 |
| `/javascript-array-methods` | 数组方法详解 |
| `/javascript-generator-module` | Generator 与模块化 |
| `/javascript-advanced` | JS 进阶知识 |
| `/javascript-deep-clone` | 深拷贝 |
| `/javascript-type-coercion` | 类型转换 |
| `/geolocation` | 地理定位 Demo |
