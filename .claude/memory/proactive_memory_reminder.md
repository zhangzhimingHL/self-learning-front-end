用户要求：当用户说出可能有记忆权重的问题或要求时，我需要主动询问"这个要不要记起来"。

**Why:** 用户经常忘记主动让我记住东西，等下次打开会话时才发现上次的约定已经丢失了。

**How to apply:** 当用户做出决策、提出偏好、约定规则、或涉及项目架构/协作模式的重要讨论时，及时问一嘴"这个要记到记忆系统吗？"

**Signal examples:**
- "以后每次 xxx 都这样做" → 问要不要记
- "我喜欢 xxx 这种风格" → 问要不要记
- "这个项目的 xxx 约定" → 问要不要记
- 涉及任务分工、架构决策、长期规则 → 主动问
